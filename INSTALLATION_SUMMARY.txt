❓ TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────

Issue: "Cannot find module 'magic-ui'"
Solution: npm install --ignore-scripts

Issue: Animations not visible
Solution: Ensure 'use client' is at top of component

Issue: Performance issues
Solution: Reduce animation duration, use viewport={{ once: true }}

Issue: Animations conflicting with CSS
Solution: Use style prop for framer-motion, CSS for base styles
